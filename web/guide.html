Hướng dẫn sử dụng hệ thống NFT Signature Checker

NFT Signature Checker là công cụ giúp bạn bảo vệ, xác thực và quản lý tài sản số (ảnh, video, tài liệu...) bằng công nghệ chữ ký số (hash) hiện đại. Dưới đây là hướng dẫn chi tiết từng bước sử dụng hệ thống:

  1. Bước 1: Chuẩn bị URL tài sản số
    - Đảm bảo bạn có đường dẫn (URL) trực tiếp đến file ảnh, video hoặc tài liệu số cần gắn hoặc kiểm tra chữ ký.
    - URL phải là đường dẫn công khai, có thể truy cập trực tiếp từ Internet (ví dụ: Google Drive chia sẻ công khai, Dropbox, Imgur, v.v.).
    - Không sử dụng link của trang web HTML, chỉ dùng link file thực (có đuôi .jpg, .png, .mp4, .pdf...).
  2. Bước 2: Gắn chữ ký NFT cho tài sản số
    - Tại trang chủ, nhập URL vào ô "Nhập URL ảnh/video".
    - Nhấn nút "Gắn chữ ký".
    - Hệ thống sẽ tự động tải file về, tạo mã hash duy nhất cho nội dung file và lưu lại bản gốc trên server.
    - Nếu tài sản đã từng được gắn chữ ký, hệ thống sẽ thông báo và không tạo lại để tránh trùng lặp.
    - Sau khi gắn chữ ký thành công, bạn sẽ nhận được thông báo kèm mã hash của tài sản số.
  3. Bước 3: Kiểm tra chữ ký NFT
    - Nhập lại URL tài sản số cần kiểm tra vào ô nhập.
    - Nhấn nút "Kiểm tra".
    - Hệ thống sẽ tải file hiện tại, tạo mã hash và so sánh với mã hash đã lưu.
    - Nếu nội dung không thay đổi, hệ thống xác nhận tài sản số còn nguyên vẹn.
    - Nếu phát hiện thay đổi (file bị chỉnh sửa, giả mạo...), hệ thống sẽ cảnh báo và hiển thị thông tin khác biệt.
  4. Bước 4: Xem lịch sử các URL đã gắn chữ ký
    - Nhấn vào mục "Lịch sử URL" trên thanh điều hướng.
    - Trang lịch sử sẽ hiển thị danh sách tất cả các URL đã được gắn chữ ký, kèm thời gian, mã hash và trạng thái.
    - Bạn có thể nhấn vào từng URL để kiểm tra lại hoặc tải về bản gốc.
  5. Bước 5: Khôi phục bản gốc tài sản số
    - Nếu phát hiện tài sản số đã bị thay đổi, hệ thống sẽ tự động gửi email cảnh báo về địa chỉ email bạn đã đăng ký.
    - Trong email cảnh báo sẽ có đường dẫn để tải lại bản gốc đã lưu trên hệ thống, giúp bạn dễ dàng khôi phục và bảo vệ bản quyền nội dung.
  6. Bước 6: Một số lưu ý khi sử dụng hệ thống
    • Chỉ sử dụng URL của các file thực sự (không dùng link HTML, trang web).
    • Đảm bảo file tải về là đúng file gốc, không bị đổi tên hoặc đổi định dạng.
    • Không chia sẻ link tài sản số cho người không tin cậy để tránh bị chỉnh sửa hoặc giả mạo.
    • Hệ thống hỗ trợ các định dạng phổ biến: PNG, JPG, JPEG, HEIC, MP4, PDF, DOCX, v.v.
    • Nếu gặp lỗi khi tải file hoặc kiểm tra chữ ký, hãy kiểm tra lại URL hoặc liên hệ hỗ trợ.
    • Để tăng bảo mật, hãy thường xuyên kiểm tra chữ ký các tài sản số quan trọng.
  7. Bước 7: Một số tình huống thực tế và giải pháp
    • Trường hợp 1: Bạn phát hiện file ảnh/video của mình bị chỉnh sửa trên Internet.
      → Hệ thống sẽ tự động gửi email cảnh báo cho bạn kèm đường dẫn tải lại bản gốc để bảo vệ bản quyền.
    • Trường hợp 2: Bạn muốn xác thực một file do người khác gửi.
      → Hãy yêu cầu họ cung cấp URL đã được gắn chữ ký trên hệ thống, sau đó kiểm tra lại để xác thực nguồn gốc.
    • Trường hợp 3: Bạn cần chứng minh quyền sở hữu tài sản số.
      → Sử dụng mã hash và lịch sử gắn chữ ký trên hệ thống làm bằng chứng pháp lý.
